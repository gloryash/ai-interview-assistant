## 1. 产品概述

### 1.1 产品名称

AI 访谈助手（AI Interview Assistant）

### 1.2 产品定位

通过 AI 语音访谈，帮助用户将模糊的想法转化为结构化文档（PRD、用户画像等）。

### 1.3 核心价值

| 价值 | 说明 |
|------|------|
| 降低表达门槛 | 语音比文字更自然，用户只需说话 |
| AI 主动引导 | 用户不需要知道文档怎么写 |
| 自动生成文档 | 访谈结束后自动产出专业文档 |
| 多场景支持 | 一套系统支持产品访谈、助教复盘等多种场景 |

## 3. 功能需求

### 3.1 功能架构

```
┌─────────────────────────────────────────────────────────────┐
│                     AI 访谈助手                              │
├─────────────────────────────────────────────────────────────┤
│  核心功能                                                    │
│  ├── 语音对话（ASR + TTS + VAD）                            │
│  ├── 场景选择（多场景切换）                                  │
│  ├── AI 访谈引导（主动提问）                                 │
│  └── 文档生成（PRD、用户画像等）                             │
├─────────────────────────────────────────────────────────────┤
│  辅助功能                                                    │
│  ├── 对话记录显示                                            │
│  ├── 文档预览（Markdown 渲染）                               │
│  ├── 文档导出（.md 文件）                                    │
│  └── 设置面板                                                │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 功能清单

| 功能 | 说明 | 优先级 | 状态 |
|------|------|--------|------|
| 语音对话 | ASR + TTS + VAD | P0 | 已完成 |
| 场景选择 | 下拉菜单切换场景 | P0 | 已完成 |
| AI 访谈引导 | AI 主动提问 | P0 | 已完成 |
| 文档生成 | 根据访谈生成文档 | P0 | 已完成 |
| 对话记录 | 实时显示对话 | P1 | 已完成 |
| 文档预览 | Markdown 渲染 | P1 | 已完成 |
| 文档导出 | 下载 .md 文件 | P1 | 已完成 |

### 3.3 核心功能详解

#### F1: 语音对话

| 项目 | 说明 |
|------|------|
| 语音识别 | 阿里云 Paraformer 实时 ASR |
| 语音合成 | 阿里云 CosyVoice TTS |
| 语音打断 | 用户说话时自动停止 AI 回复 |

#### F2: AI 访谈引导

| 项目 | 说明 |
|------|------|
| 主动提问 | AI 按预设流程引导用户 |
| 追问细节 | 对模糊回答进行追问 |
| 阶段管理 | 开场 → 核心信息 → 总结 |

#### F3: 文档生成

| 项目 | 说明 |
|------|------|
| 输入 | 访谈记录（对话历史） |
| 输出 | PRD、用户画像等文档 |
| 格式 | Markdown |

## 4. 使用流程

```
1. 配置 API Key
       ↓
2. 选择访谈场景
       ↓
3. 点击"开始采访"
       ↓
4. 语音对话（AI 引导）
       ↓
5. 点击"结束采访"
       ↓
6. 点击"生成文档"
       ↓
7. 预览/下载文档
```
